# Task

Analyze the Base blockchain transaction volume using official GitHub repositories and publicly available datasets.

## Objectives

1. Identify the official GitHub repositories related to the Base blockchain.
2. Locate any datasets, APIs, metrics, or scripts that expose:
   - Daily transaction count
   - Total transaction volume
   - TPS (Transactions Per Second)
   - Historical transaction growth
   - Active addresses
   - Gas usage
   - Block production statistics
3. Explain where each metric originates.
4. If the repositories contain scripts, explain how to execute them locally.
5. Generate Python code that downloads and aggregates transaction statistics.
6. Produce charts showing:
   - Daily Transactions
   - Monthly Transactions
   - TPS over time
   - Gas usage over time
7. Compare Base transaction volume with:
   - Ethereum
   - Arbitrum
   - Optimism
   - zkSync
8. Provide references to every GitHub file used.

## Output

Return:

- GitHub repository links
- Important file paths
- Data schema
- Python code
- SQL (if applicable)
- Example API requests
- Transaction statistics
- Visualizations
- Summary of findings

Use only official Base repositories and trustworthy public sources.
